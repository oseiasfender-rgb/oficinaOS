# OficinaOS — Limaprata Reparações Automotivas

Sistema completo de gestão para oficina de funilaria e pintura.

## Estrutura
- Frontend puro (HTML, CSS, JS)
- Arquitetura modular

## Próximos passos
- Backend Node.js
- Banco PostgreSQL
- Autenticação JWT                                                                                                                                                                                                                                                                                ## ✅ Versão Atual: Fase 3 Corrigida (v6)

### Principais Correções Aplicadas:
- Abas totalmente funcionais (Financeiro, Orçamento, Contas, Metas, Agenda, etc.)
- Resumo do orçamento (coluna direita) agora aparece corretamente
- Slider de valores visível e funcional
- Calendário com mês e ano corretos + navegação (setas Anterior / Próximo)
- Layout corrigido (conteúdo não fica mais oculto ou abaixado)
- Melhor responsividade e estabilidade geral

### Funcionalidades Principais:
- Gestão Financeira completa (receitas, despesas, KPIs, caixa x competência)
- Orçamento detalhado com peças, mão de obra, terceiros e cenários (Mínimo / Ideal / Premium)
- Agenda / Calendário profissional
- Controle de Contas a Pagar
- Gestão de Clientes e Veículos
- Controle de Materiais e Estoque
- Consultor IA integrado
- Geração de relatórios e impressão de orçamentos

### Tecnologias:
- HTML5 + CSS3 + Vanilla JavaScript
- Design responsivo com foco em uso mobile e desktop
- Suporte PWA (instalável)

### Como usar:
1. Faça o download ou clone este repositório
2. Coloque os arquivos na raiz do seu GitHub Pages
3. Acesse através do link do GitHub Pages

### Arquivos incluídos:
- `index.html` → Versão principal corrigida
- `manifest.json` → Configuração PWA
- `.nojekyll` → Para GitHub Pages funcionar corretamente
